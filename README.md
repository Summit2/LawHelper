# FindLawHelper

RAG-система для поиска ответов в Кодексе об административных правонарушениях (КоАП РФ). Система отвечает на вопросы пользователей, используя информацию из текста закона.

## Быстрый старт

### Установка
```bash
# Клонируйте репозиторий
git clone github.com/Summit2/LawHelper
cd github.com/Summit2/LawHelper

# Установите зависимости
pip install -r requirements.txt
```

### Запуск
```bash
# Основной скрипт
python main.py
```

##  Пример использования

Система загружает текст КоАП, строит поисковый индекс и отвечает на вопросы:

```python
# Пример вопроса:
"Какая статья регулирует превышение скорости?"

# Ответ системы:
"Превышение скорости регулируется статьей 12.9 КоАП РФ..."
```
